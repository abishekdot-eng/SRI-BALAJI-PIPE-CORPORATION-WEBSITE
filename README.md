# 🚀 Deployment

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/sri-balaji-pipe-corporation.git
cd sri-balaji-pipe-corporation
```

---

## 2️⃣ Push To GitHub

```bash
git init
git add .
git commit -m "Initial Commit"
git branch -M main
git remote add origin https://github.com/yourusername/sri-balaji-pipe-corporation.git
git push -u origin main
```

---

## 3️⃣ Enable GitHub Pages

1. Open Repository
2. Go to **Settings**
3. Open **Pages**
4. Select:
   - Source → Deploy from Branch
   - Branch → main
   - Folder → /root
5. Click **Save**
